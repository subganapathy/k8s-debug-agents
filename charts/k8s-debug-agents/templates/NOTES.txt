Create / rotate the Anthropic API Secret. ALWAYS strip trailing whitespace —
clipboard-pasted keys often carry a trailing newline that breaks Anthropic's
validation:

  # If the key is in your shell env:
  printf '%s' "$ANTHROPIC_API_KEY" \
    | tr -d '\n\r' \
    | kubectl create secret generic anthropic-api-key \
        -n {{ .Values.platformNamespace.name }} \
        --from-file=api-key=/dev/stdin \
        --dry-run=client -o yaml \
    | kubectl apply -f -

  # Or if the key is in macOS Keychain:
  security find-generic-password -a "$USER" -s "anthropic-api-key" -w \
    | tr -d '\n\r' \
    | kubectl create secret generic anthropic-api-key \
        -n {{ .Values.platformNamespace.name }} \
        --from-file=api-key=/dev/stdin \
        --dry-run=client -o yaml \
    | kubectl apply -f -

credential-authz's fsnotify loader picks up the rotation within a few seconds
of kubelet's projected-Secret sync (~60s worst case). No pod restart needed.
